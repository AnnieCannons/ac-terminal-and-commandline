What is a Shell?

The terminal is an interface in which you can type and execute text based commands.

The command for changing directories

cd

cd means "change directory."

parent-to-child: cd [folder name]

child-to-parent: cd ..

Changing Directories

  • cd <relative path>
  • cd <absolute path>
  • cd <symbolic path>
  • -> cd ../exampleDir
  • -> cd /exampleDir/newFolder
  • -> cd ~/exampleDir
../ is based on this folder you're currently in, navigate up to its parent, then navigate down into example directory.absolute path - tell Terminal where we want to end up based on where we are now. symbolic path is when we use a character like a ~ to reference a starting point; the ~ references the user's home folder in this example.

Common Commands in Terminal

Tool Action
ls List what is in the current directory.
cd Change directory.
mkdir Make a new directory.
touch Create or update an existing file's timestamp. The touch command expects the name of your new file as an argument.
mv Move a file from one place to another. See Docs.

Flags

Some commands have additional options that allow flags.

Flags are preceded by a hyphen, and are often a single character.

ls -a = show hidden files

ls -l = use a “long list” format (shows more verbose output)

Terminal Commands

  • The left/right arrow keys let you edit within a command
  • The up/down arrow keys let you select previous commands
  • CTRL+A goes to beginning of command, CTRL+E to the end.
  • The tab auto-completes file names! Best friend!
